# Sources

This repository is a clean-room open-source implementation.

Public-facing contents were written fresh for this package and validated against the release audit.

Included sources:

- original package code in `src/script_to_video_production_agent/`
- original repository documentation in this root and `references/`
- synthetic fixture content in `assets/fixtures/`
- locally authored tests and eval definitions in `tests/` and `evals/`

Excluded by policy:

- client names
- private local paths
- private scripts
- private source prompts
- private generated assets
- inherited Git history

Third-party libraries:

- Python standard library
- `python-docx`
