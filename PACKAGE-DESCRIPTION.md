# Package Description

Script-to-Video Production Agent is a local workflow system for turning tutorial and explainer scripts into narration-locked scene plans, reviewable visual direction, prompt documents, and resumable asset queues.

It is built for operators who want:

- script import from DOCX, Markdown, or plain text
- exact narration preservation
- human-reviewed visual cleanup before production
- vendor-neutral output with optional InVideo formatting
- still and video prompt packs
- queue tracking with retries and approvals
- clean-room packaging for public release
