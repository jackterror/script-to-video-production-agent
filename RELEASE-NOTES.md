# Release Notes

## Version 0.1.0

Initial public candidate release.

Highlights:

- shared scene schema across import, review, prompting, and queue stages
- narration hashing for integrity checks
- DOCX, Markdown, and JSON exports
- vendor-neutral builder prompts and InVideo-oriented prompt profile
- still-image, video-shot, and image-only prompt exports
- SQLite-backed queue with retry and approval tracking
- clean-room audit and release packaging scripts
- synthetic fixtures and local unit tests
