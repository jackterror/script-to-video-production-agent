# With-Skill vs Baseline

Use this repository to compare:

- baseline agent scene extraction from raw scripts
- skill-guided scene extraction with narration hashing
- baseline visual cleanup without review traceability
- skill-guided review export and apply cycle
- baseline prompt drafting without queue state
- skill-guided prompt export plus SQLite queue tracking

Success criteria:

- narration preserved exactly
- scenes ordered consistently
- review changes traceable
- queue retries and approvals deterministic
